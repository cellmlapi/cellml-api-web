--- title: CellML API Applications description: Applications using the CellML API ---

OpenCOR

OpenCOR is a cross-platform modelling environment which can be used on Windows, Linux and OS X to organise, edit, simulate and analyse CellML files. Read more...

CellMLSimulator

CellMLSimulator is a tool that lets you generate results and figures to put into publications starting from a CellML file and metadata about the graphs and simulations. Read more...

OpenCMISS

OpenCMISS is a mathematical modelling environment that enables the application of finite element analysis techniques to a variety of complex bioengineering problems. Read more...

PMR2 / CellML Model Repository

PMR2 is an open source model repository software project. It is used to run the CellML Model Repository, a repository containing hundreds of CellML models. Read more...

Antimony

Antimony is a modular human-readable, human-writeable model definition language which can be converted to and from CellML and SBML Read more...
