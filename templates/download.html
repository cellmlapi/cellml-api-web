Snapshot builds and the latest source

There are several different ways to get the API. Snapshot builds are produced automatically once a day on days when the CellML API was changed, but only if the automated tests of the API pass. These builds refer to themselves as version $nextplanned$, which is the version we are currently working towards. While they may not have been tested as much as a release, they have passed automated tests, have all the latest improvements to the API, and so are the best option for most developers building on the API.

You can find the latest snapshots here. You might also be interested in browsing or obtaining the very latest API source code.
